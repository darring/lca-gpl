Generic Error Logger

This class handles the generic error logging functionalities for the steward that aren't handled elsewhere in the daemon.

Specifically, this deals with logging of errors defined in the errno(3) Linux header file, which are very generic errors.

Usage

The Error Logger is created using the ErrLogger::ErrLogger(StewardLogger *logger) constructor.
