Logger for the EIL Client Agent Steward

Introduction

This is the class which handles the logging for the EIL Linux Client Agent.

Details

This logging class is intended to provide an abstracted interface which wraps multiple ways to log to system files. It can be used in one of two ways:

Dual system and private logging

This is the standard way to envoke and create the logging class instance. When created this way, the class instance will log to a specified file (a private log) as well as to syslog. This allows for a private log which can be retained, moved, etc. and which only contains the log details for this daemon. But at the same time, we will also be logging to a system-wide log which will allow for external monitoring software which may be aggregating data across multple systems to collect the log information from our daemon.

To envoke this logger class in this way, you must call StewardLogger::StewardLogger(char *logFile);
