Quickly log an entry.

Call this when you wish to quickly log an entry without having to call begin/end. Note that this does NOT eliminate the steps required to enable logging, it simply masks them. This log method should only be used when a quick one-liner is to be logged, not when there are extended log entries to be written.

void StewardLogger::SetPriority ( int  priority  ) 

Set the priority for syslog.

This sets the priority for future syslog calls. Rather than doing priority changes on each call to log entries, we decided to just set them periodically as needed (since, in our case, they wont be changing much). Might not be the best in practice, but we'll find out as this thing gets more widely used :-)

Note:
The default priority for this logging class will be LOG_INFO
Parameters:
priority Bitwise ORed syslog priority level
