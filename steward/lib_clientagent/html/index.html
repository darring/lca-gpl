ClientAgentHelper Interface

Introduction

This class provides the interface to the client agent helper scripts provided by the dispatcher.

Usage

Once you have a ClientAgentHelper instance, use the ClientAgentHelper::Get(char *result, int size, ClientAgentOptions option) method to interact with the external client agent helper script.
